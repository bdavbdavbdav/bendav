program note:

In medias res.

For Nick Meryhew is an ongoing work for trombonist and MIDI trombone that will ideally never be completed. This work builds a semi-open form that can be configured numerous ways by the performer. There is no preference for any given form as long as the basic rules of configuration are followed.
